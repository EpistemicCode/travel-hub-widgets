How to embed

Three steps, zero code

  1. Copy any widget URL below (click the Copy button).
  2. Paste the URL into an embed block on your page:
    • Notion: type /embed, paste, hit Embed link.
    • Website / blog: use an <iframe src="…"> with the widget's width & height.
    • Ghost / WordPress / Substack: use the HTML or Embed block, paste the iframe.
  3. Size the embed to match the dimensions shown on each card.
